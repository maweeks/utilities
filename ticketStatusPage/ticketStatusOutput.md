## Key

✅ Done
🟣 In test
🟡 Ready to pick up / in progress
⛔ Defined/started but blocked from progress
📝 Not ready to pick up / not yet defined
❔ Status unknown 😨
ABC-_____ - No ticket created yet

❔ [ABC-](https://abc.atlassian.net/browse/ABC-) - Description
